利用可能な要素

ePub 2.0 で利用可能な要素を紹介します。

構造要素

html, head, title, body 要素が使えます。HTML の基本ですね。

テキスト

ヘッダー

h1, h2, h3, h4, h5, h6 のヘッダー要素が使えます。このテキストでは、h1, h2, h3 を使っています。

段落

p 要素を使って段落を作ります。

段落内で改行が必要な場合は br 要素が使えます。

例:

  段落の中で<br/>
  改行

結果:

段落の中で
改行

引用

blockquote 要素を使って、引用ができます。

三つの指輪は、空の下なるエルフの王に、
七つの指輪は、岩の館のドワーフの君に、
九つは、死すべき運命の人の子に、
一つは、暗き御座の冥王のため、
影横たわるモルドールの国に。

一つの指輪は、すべてを統べ、
一つの指輪は、すべてを見つけ、
一つの指輪は、すべてを捕らえて
くらやみのなかにつなぎとめる。
影横たわるモルドールの国に。

q 要素を使って、インライン引用もできます: 一つの指輪は、すべてを統べ

cite 要素を使って、引用元を表現することもします: 指輪物語より引用

そのまま

pre 要素を使って、そのまま表示することが可能です。

#define MAX 100
int main(void)
{
  int sum = 0;
  for (int i=0; i<MAX; i++)
  {
     sum += i;
  }
  return 0;
}

強調

より強い強調に strong 要素を、普通の強調に em 要素が利用可能です。

その他

特に意味のないブロック要素に div、インライン要素に span 要素が使えます。

意味のある単語

  • dfn 要素: 定義を表す
  • code 要素: int i=0 の様にプログラム・コードを表す
  • kbd 要素: C-x C-s の様にキーボード入力を表す
  • samp 要素: samp 要素の様にサンプルを表す
  • var 要素: PATH の様に変数を表す

省略文字

abbr 要素を使って省略文字を表現できます: WHO

acronym 要素も利用可能です: HTML

ハイパーリンク

a 要素も利用可能です。「構造要素」へジャンプしてみましょう。

リスト

番号なしリストの ul と、番号ありリストの ol、そして定義リストの dl が利用可能です。

説明省略

object, bdo, area, map, meta, style, link, base の説明は省略します。

b, i, tt, big, small 要素の利用は控えて CSS を使いましょう。

上付き1や下付き2 は必要に応じて使いましょう。

hr 要素はよく分からんです。
